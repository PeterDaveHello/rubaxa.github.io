b-toolkit.js и Радары

b-toolkit.js

b-toolkit.js

— это «минималистичная» обвязка для fest блоков и написания кросс-проектных «ui компонентов».

Создание блока

			var block = toolkit["b-toolkit"].extend({
				init: function (){
					// Описание логики инициализация блока
					this.$el.toggleMod("inited", true); // как пример
				},
				destory: function (){
					this.$el.toggleMod("inited", false);
					toolkit["b-toolkit"].fn.destory.call(this);
				}
			});

Базовые свойства

  • el — ссылка на DOM-элемент
  • $el — ссылка на jQuery-коллекцию
  • template — шаблон блока, либо блоков
  • templateBlock — имя блока
  • eventEmitTo — делегировать события на другой контекст
  • eventEmitPrefix — добавить к названию события префикс

Базовые методы

  • __lego() — конструктор блока (лучше не трогать)
  • init() — инициализация (для переопределениея)
  • on(event, fn) — подписаться на событие
  • off(event, fn) — отписаться
  • emit(type, args) — испустить событие
  • $(selector) — эквивалент this.$el.find(selector)
  • bound(nameOrFn) — привязать метод к this

Базовые методы

  • setData(data) — установить дату
  • getData() — получить установленные данные
  • getHtml([data]) — получить html блока
  • render() — render блока, см. getHtml()

Усложняем I

			var dropdown = new toolkit["b-dropdown"]({
				el: "#my-dropdown",
				eventEmitTo: mySuperView,
				eventEmitPrefix: "dropdown"
			});
			// Рендерим блок
			dropdown.render();

			});

			toolbar.render();
